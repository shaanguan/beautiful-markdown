# Changelog

## v1.1 — 2026-05-28

- **新增** 识别颜色和文字颜色:支持 `{c:#0089FF}文字{/c}`、`{bg:#fff59d}文字{/bg}` 等简写,以及 KaTeX `\color{...}{...}` 兼容。
- **新增** Bullet Point 展开与折叠:含子列表的列表项左侧出现折叠箭头,点击收起/展开。
- **新增** 自动识别文本语言:加载文档后扫描内容,判断中文 / 其他语言。
- **新增** 翻译目标语言自动设置:根据识别到的原文语言自动选择目标——中文 → 英文,其他 → 中文。
