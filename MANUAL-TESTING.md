## 测试前准备

1. **重载扩展**：`chrome://extensions` → Beautiful Markdown → **重新加载**（每次拉代码后必做）。
2. **建议测试文件**（仓库内）：
  - `fixtures/markdown-renderer-test.md` — 五项渲染回归（图/高亮/上下标/emoji/RTL）

3. **翻译**：Settings 中填好有效 API Key（仅 D 类需要）。
4. **记录**：`PASS` / `FAIL` + 浏览器版本 + 是否刚重载扩展。

## G. 5 分钟冒烟（建议每次发版跑一遍）

- `file://` 打开长文 → 滚到中部 → **Split** → 右栏打开另一文件 → 两栏各滚一段 → 切回 **标准** → 左栏位置仍在文中部
- 点扩展图标 → **确认空态无 Switcher / 无 doc-tools** → 粘贴或打开 md → chrome 出现 → **Split** → 右栏选文件
- 翻译短文 → **双栏对照** → 右栏换文件为普通 `.md`
- 编辑页：改几个字 → 撤销/重做 → 保存 → 阅读页更新
